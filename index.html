Data syncs to your GitHub repository · Nothing leaves your browser except direct calls to the GitHub API

Settings

Ledger stores your data as a JSON file in a GitHub repository so it follows you across devices. Use a private repo for this data — it can be different from the (public) repo that hosts this site. Create a fine-grained Personal Access Token scoped to only that repo with read/write access to "Contents".
Create a token at github.com/settings/personal-access-tokens/new → pick the data repo → permissions → Contents: Read and write. The token is stored only in this browser's local storage, never committed anywhere.
